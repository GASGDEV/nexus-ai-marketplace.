# Nexus AI Marketplace
> Projeto Final - Desenvolvimento Web (Sistemas de Informação)

## Sobre o Projeto
Um mini-ecommerce focado na venda de assets para Inteligência Artificial (prompts, modelos e datasets).

## Tecnologias
- HTML5 Semântico
- CSS3 (Custom Properties & Flexbox/Grid)
- JavaScript (Client-side validation)
- PHP & MySQL (CRUD completo)

## Estrutura de Páginas (Etapa 1)
1. **Home**: Vitrine de produtos em destaque.
2. **Catálogo**: Lista completa de modelos de IA.
3. **Contato/Pedido**: Formulário de solicitação customizada.
